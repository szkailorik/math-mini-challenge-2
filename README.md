# 数学巅峰收束段 · Math Mini Challenge 2

小学高年级双人数学训练系统第二代 —— 纸笔闭环 · 拟真考卷 · 永久错题本。

上一代（[math-mini-challenge](https://github.com/szkailorik/math-mini-challenge)）完成了第一阶段计算流利度训练。两位学员在乘法、除法、分数运算（旧卷大题一、二、五）上已经熟练，本项目是为「下一阶段」重新设计的训练引擎。

## 训练理念：为什么重做

已熟练的题型继续大量刷是低效的。本阶段把它们降级为每日 6 题的**核心保温**抽检，主训练火力转向四个真正决定初中衔接的能力：

| 大题 | 训练域 | 分值 | 定位 |
|---|---|---|---|
| 一 | 口算冲刺（限时 3 分钟） | 12 | 速度与稳定度 |
| 二 | 核心保温 | 12 | 已过关题型抽检防回退 |
| 三 | 表征互通 | 18 | 分数↔小数↔百分数↔比 自由切换 |
| 四 | **复杂四则混合** | **25** | **主战场：先看结构选方法** |
| 五 | 单位 · 率 · 关系 | 16 | 量纲统一与建模 |
| 六 | 方程与简算 | 17 | 未知数位置 + 结构眼 |

满分 100 分，建议 40 分钟，拟真期末卷版面（得分栏、分值标注、限时提示）。

## 核心特性

- **双人双卷**：KAI（进阶卷 L3）与 Lorik（提高卷 L2）同一套号各自适配难度，一键打印双人四页。
- **确定性生成**：同一套号永远生成同一份试卷（种子化随机），可随时补打，无需缓存。
- **拟真考卷**：A4 精确分页、考试头部、大题分值、答题留白按题型自适应。
- **永久错题本**：批阅时标记 ✗错/△粗心/？需讲解 自动入册；间隔复练调度（重错 1 套间隔、普通 4 套间隔）；重错题需两次确认才算掌握。
- **错题复练卷**：原题回炉 + 同类变式 1:1 自动配对，附独立答案页。
- **家长批阅闭环**：紧凑答案批阅页（答案 + 一句检查口令 + 勾选框）→ 系统一键录入（默认全对只点错题）→ 讲解清单（只列复错与需讲解题）。
- **仪表盘**：双人各训练域近 6 套正确率、错题存量、晋级判定（≥10 套 + 复杂混合 ≥90% + 无到期错题）。
- **零依赖零构建**：纯静态 ES Modules，GitHub Pages 直接部署；数据存本机 localStorage，支持 JSON 导出/导入迁移。

## 每日 25 分钟家长流程

1. 「试卷」→ 打印双人四页（每人 2 页 A4）+ 答案批阅页
2. 孩子限时完成（大题一 3 分钟，整卷 40 分钟内）
3. 家长纸上批阅（✓/△/✗/？），答案旁的提示就是检查口令
4. 「批阅」页录入（默认全对，只点非✓的题）
5. 每 2-3 天打一张「到期复练卷」，做完回写结果
6. 周末打「讲解清单」，只讲复错与需讲解的题

## 开发

```bash
npm run validate   # 60 套 × 2 学员生成器校验（确定性、数值卫生、覆盖骨架）
python3 -m http.server 8000   # 本地预览（ES Modules 需要 http）
```

推送 main 分支自动校验并部署到 GitHub Pages。
