# 一页纸日课系统 · 设计文档

日期：2026-07-11 ｜ 状态：用户已确认（"椰子系统"）｜ 依据：训练量调研（单次同类>3题无长期增益、交错12题结构 d≈0.83、10-12岁注意窗口20-30分钟、失分率排序：简算53.6%>多位笔算>四则混合60.4%>分数除法）

## 原则（奥卡姆）

因果链：做题→暴露错误→纠正→隔天再测→巩固。不服务此链的部件后台化。
正反馈三开关：和自己比（口算纪录）、看得见的毕业（掌握章）、以胜利收尾（必赢题）。
红线：**永不用加题惩罚错误**（错→明天仍 3 题）。

## 每日一张 A4（每人，结构永不变）

```
① 口算冲刺 40 题 · 3 分钟（页眉含 上次成绩 与 最佳纪录）
② 主攻难点 3 题（唯一主攻位）
③ 复现主题 A × 2 题 + 复现主题 B × 2 题
④ 错题二过 2 题（到期错题原题回炉，无到期则省略）
⑤ 必赢收官 1 题（已掌握内容中取，保证以胜利结束）
页脚正反馈条：连续第 N 天 · 口算最佳 X 秒 · 掌握章 N 枚 · 本周主攻：XX
```

主题类合计 ≤10 题；总时长 ~20 分钟。**周六比赛日**：手动按钮出完整拟真卷（v2 100分卷保留原样）。日常界面不再有模式切换。

## 主题体系（TOPICS，7 个）

按失分率排序的主攻队列：smart 简便运算 → mixed 复杂四则混合 → mixfd 分数小数混合 → fracdiv 分数除法/分数两步 → bigcalc 多位笔算(小数乘除) → equation 方程 → unitrate 单位率。
题目来源：v2 域生成器按 tag 前缀过滤（确定性超采样后筛选），难度沿用 adaptive 的动态 level。

## 轮换规则（准确率驱动）

- 主攻位：队列中第一个未毕业主题；**连续 2 个批阅日主攻区 3/3 全对 → 毕业**（发一枚掌握章），下一个主题顶上；队列走完从头轮（保温强度）。
- 复现位：其余主题按套号轮转（round-robin），每天 2 个；已毕业主题照常进复现（防回退）。
- 错题位：现有间隔调度取到期前 2 条，原题回炉，结果回写错题本。
- 必赢位：维持态口算技能或已毕业主题中出 1 道基础题。

## 数据与确定性

- `MMC2_attack_{student}`：{ topicKey, streakDays, graduated: [topicKey...] }（纳入 Gist 同步，合并取 graduated 并集 + streak 取新）
- 口算最佳成绩：saveSprintScore 增加 best 字段（correct≥38 时按秒数取优）
- 连续天数：从 history 日期序列计算（缺一天断，比赛日不计断）
- 盖章：stamp 增加 onepage 字段（attackTopic/reviewTopics/errorIds/winnerRef），打印/批阅时固化
- 单页 A4 硬约束：内容底 ≤276mm

## 批阅（30 秒）

批阅页 = 口算区（点错+秒数，现有）+ 主题区 10 题（点错）+ 提交。提交链：口算→mastery；主题题→recordGrades（进错题本+主攻统计）；错题位→recordPracticeResults；更新 attack 状态与掌握章。

## 保留与移除

- 保留：拟真卷（比赛日按钮）、错题本快速出册、仪表盘、同步、口算 mastery
- 移除日常存在感：日课/拟真模式切换按钮（日课即默认，比赛日是动作不是模式）
